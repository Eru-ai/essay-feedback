# Essay Feedback Tool

NCEA Level 2 English essay feedback using the Gemini API.

## Setup

1. Create and activate the virtual environment (see below).
2. Install dependencies: `pip install google-generativeai python-dotenv`
3. Add your Gemini API key to `.env`
4. Paste your essay into `essay.txt` and your source text into `source.txt`
5. Run: `python essay_feedback.py`
